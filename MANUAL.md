# 📖 Buku Panduan Penggunaan: Cegah Anemia Yuk!

Aplikasi "Cegah Anemia Yuk!" adalah platform edukasi interaktif berbasis web (PWA) yang dirancang untuk membantu remaja putri memahami dan mencegah anemia melalui modul belajar, pelacakan harian, dan evaluasi pengetahuan.

---

## 📱 1. Panduan untuk Siswa (Responden)

### A. Cara Masuk ke Aplikasi
1. Buka browser di HP Anda (Chrome/Safari).
2. Masukkan URL: `https://anemia-tracker.vercel.app/`.
3. Masukkan **Nama Lengkap** dan **6 Digit Kode Unik** yang diberikan oleh kakak peneliti.
4. Klik **"Mulai Petualangan! 🚀"**.

### B. Alur Belajar yang Benar
Untuk mendapatkan hasil maksimal, ikuti langkah ini:
1. **Pre-Test**: Kerjakan 7 soal awal untuk melihat sejauh mana pengetahuan Anda. Jangan takut salah! 🎯
2. **Belajar Modul**: Baca 6 modul seru (dari Pengertian sampai Aksi Nyata).
3. **Misi Harian (Daily Mission)**:
   - Centang **"Minum TTD hari ini"** setiap kali Anda meminum Tablet Tambah Darah.
   - Jawab **"Soal Harian"** untuk menambah poin Anda. 🔥
4. **Post-Test**: Setelah semua modul selesai, kerjakan kuis akhir untuk melihat peningkatan skor Anda! 🌟

### C. Mengatur Profil & Avatar
- Klik tab **Profil** di pojok kanan bawah.
- Pilih **Avatar** (Super Hero/Icon) yang paling menggambarkan diri Anda.
- Ganti **Hari Jadwal TTD** sesuai instruksi (contoh: setiap Senin).

---

## ⚙️ 2. Panduan untuk Admin (Peneliti)

### A. Mengakses Panel Admin
1. Di halaman Login, klik tulisan kecil **"🔒 Akses Portal Admin"** di bagian paling bawah.
2. Masukkan Password Admin (Default: `nelysa123`).
3. Klik **"Buka Dashboard"**.

### B. Fitur Dashboard Admin
- **Ringkasan (Overview)**: Melihat total responden, rata-rata kenaikan skor (Pre vs Post), dan persentase kepatuhan minum TTD dalam bentuk grafik.
- **Data Responden**: Tabel detail yang berisi nama siswa, skor tes, dan berapa kali mereka sudah minum TTD.
- **Manajemen Kode**: 
  - Gunakan **"Generator Kode"** untuk membuat kode baru secara otomatis. 
  - Bagikan kode ini ke calon responden agar mereka bisa mendaftar.

### C. Menarik Data (Export CSV)
- Di tab **Data Responden**, klik tombol **"Excel/CSV"** (ikon download).
- File akan terunduh dan bisa dibuka menggunakan Microsoft Excel untuk diolah lebih lanjut dalam penelitian.

---

## 🛠️ 3. Cara "Mendownload" Aplikasi (PWA)

Aplikasi ini menggunakan teknologi **PWA (Progressive Web App)**. Anda tidak perlu mencari APK di PlayStore. Cukup ikuti cara ini agar muncul di layar HP:

### Untuk Android (Google Chrome)
1. Buka URL aplikasi di Chrome.
2. Klik **ikon titik tiga (⋮)** di pojok kanan atas.
3. Pilih **"Tambahkan ke Layar Utama"** atau **"Instal Aplikasi"**.
4. Selesai! Ikon aplikasi akan muncul di layar HP seperti aplikasi biasa.

### Untuk iPhone/iOS (Safari)
1. Buka URL aplikasi di Safari.
2. Klik tombol **Share (ikon kotak dengan panah ke atas ↑)** di bagian bawah.
3. Scroll ke bawah dan pilih **"Add to Home Screen"**.
4. Klik **"Add"**. Aplikasi akan terpasang di layar utama iPhone Anda.

## 🆘 Troubleshooting (Tanya-Jawab)
- **Data Tidak Muncul?**: Coba klik tombol **Refresh** (ikon panah melingkar) di header aplikasi.
- **Lupa Password Admin?**: Hubungi pengembang untuk reset password melalui database Supabase.
- **Aplikasi Lambat?**: Pastikan koneksi internet stabil saat pertama kali login atau saat melakukan sinkronisasi data Supabase.
